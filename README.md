# taxonomylist
Shortcode implementabile su wordpress che mostra le subcategory in archivio (custom taxonomy) associate con i custom post types

#Note ed accorgimenti per l'installazione
<li>Inserire il contenuto del file functions.php in coda al file functions.php del proprio tema in uso, alternativamente creare un child per il tema in uso e quindi inserire il file functions.php nella cartella child (consigliato)</li>
